# Dotfiles

Creates the following dotfiles:

- `.ackrc`
- `.vimrc`
- `.gvimrc`
- `.tmux.conf`
- `.vim/`
- `.gitignore`
- `.gitmodules`

## Installation

- `git clone git@github.com:wesbaker/dotfiles.git`
- `bundle install`
- Make sure the various dotfiles this project creates don't exist
- `rake install`
