# Dotfiles

Creates the following dotfiles:

- `.ackrc`
- `.vimrc`
- `.gvimrc`
- `.vim/`
- `.gitignore`
- `.gitmodules`

## Installation

- [Install Homebrew](http://brew.sh)
- `brew tap homebrew/bundle`
- `brew bundle`
- `git clone`
- `git submodule init` and `git submodule update`
- Make sure the various dotfiles this project creates don't exist
- `rake update`
